Welcome to Metrics Editor

The main goal of this webapp is the possibility to create, edit and delete metrics in a visual way.


For doing something related to this metrics anyone has to go to Metrics Editor section.


In Metrics Editor section anyone can find a table with all the metrics created until the moment. Every metric can be shown in detail, edited and deleted.

Moreover, this table has the possibility to filter and sort metrics.


During the creation of every metric a couple of files are generated which can be shown in the detailed information of it.


For a better understanding about the fields that take part in the creation of any metric, please go to Help section.
